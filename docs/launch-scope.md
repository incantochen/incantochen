# launch-scope.md — 上線必要子集（Launch-Critical Scope）

> 文件更新日期：2026-07-14
> 用途：定義「距離收單上線還差什麼」——把未完成任務分為 A（硬性）／B（體驗必要）／C（決策擋路）／D（上線後可修）四級，取代「看 tasks.csv 全表估距離」的失真讀法。
> 權威分工：任務本體仍以 `docs/tasks.csv` 為準（A/B 級任務標題已加 🚀 記號）；本檔管「分級與決策期限」。
> 維護原則：新任務登記時順手分級；C 級每項有決策期限，到期未拍板＝自動升級為阻塞警報；上線後本檔封存。
